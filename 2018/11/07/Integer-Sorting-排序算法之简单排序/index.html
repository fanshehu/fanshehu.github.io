Integer Sorting - 排序算法之简单排序

发表于 2018-11-07 | 分类于 Algorithm

本文介绍三种简单排序，算法复杂度皆为$O(n^2)$.

Selection Sort - 选择排序

每有一个新的数字，插入到已排好序列中该在的位置

Bubble Sort - 冒泡排序

从左到右，顺序相反的就交换，只到所有顺序一致

Insertion Sort - 插入排序

每有一个新的数字，插入到已排好序列中该在的位置
